## 3. Редактор сайта — главный инструмент

**Внешний вид → Редактор** — это главное место, где вы редактируете дизайн и контент страниц.

### Что можно редактировать в Редакторе сайта

```
Редактор сайта
├── Шаблоны (Templates)
│   ├── Главная страница (Front Page) — главная страница сайта
│   ├── Страница (Page) — шаблон обычных страниц
│   ├── Запись (Single) — шаблон статей
│   └── Архив (Archive) — шаблон списка статей
└── Части шаблона (Template Parts)
    ├── Header — шапка с логотипом и меню
    └── Footer — подвал сайта
```

### Как открыть редактор

1. Зайдите в **Внешний вид → Редактор**
2. В левой панели выберите **Шаблоны** или **Части шаблона**
3. Нажмите на нужный шаблон/часть — откроется редактор

### Интерфейс редактора

- **Центральная область** — визуальный предпросмотр страницы
- **Левая панель** — навигация по шаблонам
- **Правая панель** — настройки выделенного блока
- **Верхняя панель** — кнопка **Сохранить** (синяя кнопка)

> **Важно:** Изменения вступают в силу только после нажатия кнопки **Сохранить** в правом верхнем углу.

### Как выделить блок для редактирования

1. Кликните на элемент в центральной области
2. Над ним появится название блока (например, «Site Header» или «Hero»)
3. Справа откроются его настройки

## 4. Как редактировать Шапку сайта (Header)

Шапка — это верхняя полоса с логотипом и навигационным меню.

**Путь:** Редактор сайта → Части шаблона → **Header**

### Что можно изменить

После клика на блок шапки, в **правой панели** появятся вкладки:

#### Логотип

| Поле | Описание |
|------|----------|
| **Dark logo** | Тёмный логотип (отображается на белом фоне) |
| **Light logo** | Светлый логотип (отображается на тёмном фоне) |
| **Alt text** | Текст-замена для логотипа (для поисковиков и читалок) |

Чтобы поменять логотип — нажмите кнопку выбора изображения, загрузите новый файл или выберите из библиотеки медиафайлов.

#### Навигационное меню (Navigation Menu)

Меню состоит из **пунктов верхнего уровня**, у каждого могут быть **подпункты** (dropdown) и **под-подпункты**.

Каждый пункт меню имеет поля:

| Поле | Описание |
|------|----------|
| **Label** | Текст пункта меню (например, «Bónus») |
| **URL** | Ссылка (например, `/bonus-de-boas-vindas/` или внешняя ссылка) |
| **Icon** | Путь к иконке (например, `assets/img/2020/12/menu_bonus-3.svg`) |
| **Badge text** | Маленький бейдж на пункте (например, «novo») |
| **Badge bg color** | Цвет фона бейджа (например, `#12a96a`) |
| **Badge text color** | Цвет текста бейджа |
| **New badge** | Переключатель — показывать ли бейдж «NEW» |
| **Mega menu** | Переключатель — использовать ли мегаменю для этого пункта |
| **Mobile 2nd level** | Переключатель — показывать ли пункт на 2-м уровне в мобильном меню |

Для каждого пункта есть секция **Children** — подпункты. Нажмите **Add sub-item** чтобы добавить, или **Remove** чтобы удалить.

> **Совет по URL:** Внутренние страницы указывайте как `/путь/` (начиная со слэша). Внешние ссылки — полный адрес `https://example.com/`.

## 5. Как редактировать Подвал сайта (Footer)

**Путь:** Редактор сайта → Части шаблона → **Footer**

Кликните на блок подвала. В правой панели:

| Поле | Описание |
|------|----------|
| **Logo** | Логотип в подвале |
| **Alt text** | Альтернативный текст логотипа |
| **Disclaimer text** | Текст о jogo responsável (ответственной игре) |
| **Copyright** | Строка копирайта (например, `© 2026 – Aposta Legal`) |

#### Группы ссылок

| Группа | Описание |
|--------|----------|
| **Info Links** | Ссылки колонки «Informações» (Jogo Responsável, Os Nossos Critérios) |
| **More Links** | Ссылки колонки «Saber Mais» (Contactos, Sobre Nós) |
| **External Links** | Ссылки на сайты других стран (Чили, Эквадор, Мексика...) — с флагом |
| **Footer Links** | Ссылки в самом низу (Политика конфиденциальности, Условия) |

Каждая ссылка в группах имеет поля:
- **label** — текст ссылки
- **url** — адрес страницы
- **image** (только для External Links) — флаг/иконка страны

## 6. Как редактировать Главную страницу

**Путь:** Редактор сайта → Шаблоны → **Главная страница (Front Page)**

Главная страница состоит из набора блоков сверху вниз. Кликайте на нужный блок и редактируйте его настройки в правой панели.

### Структура главной страницы

```
┌─ Age Disclaimer (строка +18)
├─ Site Header (шапка)
├─── Page Main (обёртка страницы)
│    ├─ Hero (баннер с H1)
│    ├─ Offer List (список казино)
│    ├─ Two Column Text (текстовые блоки)
│    ├─ About List (списки с галочками)
│    ├─ Data Table (таблицы)
│    ├─ Info Grid (информационные блоки)
│    ├─ Related Links (карточки ссылок)
│    ├─ Live Odds (виджет ставок)
│    └─ News Slider (слайдер статей)
└─ Site Footer (подвал)
```

## 7. Управление Офферами казино

**Путь:** Левое меню → **Casino Offers**

Это список всех казино, которые отображаются на главной странице в секции «CASAS COM LICENÇA».

### Как добавить новый оффер

1. Нажмите **Добавить новый Casino Offer**
2. В поле **Заголовок** введите название казино (например, «Betano»)
3. Заполните поля в блоке **Casino Offer Details**:

| Поле | Описание | Пример |
|------|----------|--------|
| **Logo** | Логотип казино (картинка) | — выберите файл |
| **Rating** | Рейтинг от 0 до 5 | `4.8` |
| **Bonus** | Текст бонуса | `100% até 500€` |
| **Features** | Преимущества (по одному на строку) | `Levantamentos rápidos` |
| **Payment icons** | Иконки способов оплаты (несколько картинок) | — выберите файлы |
| **Offer URL** | Ссылка для кнопки «Jogar» | `https://apostalegal.pt/ir/betano` |
| **Review URL** | Ссылка для кнопки «Análise» | `/betano-portugal/` |
| **License status** | Статус лицензии | `SRIJ n.º 017/018` |
| **Legal offer** | Чекбокс — легальный ли оффер | ✓ |

4. Справа найдите блок **Атрибуты страницы** → поле **Порядок** — это число определяет позицию казино в списке (1 = первое, 14 = последнее)
5. Нажмите **Опубликовать**

### Как изменить порядок казино

1. Откройте нужный оффер
2. В правой панели найдите **Атрибуты страницы → Порядок**
3. Введите число (чем меньше — тем выше в списке)
4. Нажмите **Обновить**

### Как временно скрыть казино

1. Откройте оффер
2. В правой панели найдите **Статус** → переключите на **Черновик**
3. Нажмите **Обновить**

Казино пропадёт с сайта, но данные сохранятся.

## 8. Статьи и Новости

**Путь:** Левое меню → **Записи**

### Как опубликовать статью

1. Нажмите **Добавить новую запись**
2. Введите заголовок в верхнем поле
3. Напишите или вставьте текст статьи в основной области
4. В правой панели:
   - **Изображение записи** — главное фото статьи (отображается в слайдере)
   - **Рубрики** — категория статьи (например, «Crash», «Casino Betano»)
   - **Метки** — теги
5. Нажмите **Опубликовать**

### Как обновить слайдер новостей

Если слайдер News Slider настроен на **Ручной режим (Manual)** — нужно обновить список статей в блоке вручную через Редактор сайта.

Если слайдер настроен на **Автоматический режим (Query)** — новые опубликованные статьи появятся автоматически.

## 9. Справочник блоков

### 🔲 Age Disclaimer

**Что делает:** Полоса в самом верху страницы с предупреждением «+18».

| Параметр | Описание | Пример |
|----------|----------|--------|
| **Text** | Основной текст | `JOGA COM RESPONSABILIDADE` |
| **Link text** | Текст ссылки | `INCLUI CONTEÚDO COMERCIAL` |
| **Link URL** | Адрес ссылки | `/sobre-nos/` |

---

### 🖼️ Hero (Баннер/Героя)

**Что делает:** Первый большой блок страницы с фоновым изображением и заголовком H1.

| Параметр | Описание | Пример |
|----------|----------|--------|
| **Title** | Заголовок H1 — главный заголовок страницы | `Casinos Online Legais em Portugal` |
| **Text** | Подзаголовок/описание под H1 | `Comparamos o número de jogos...` |
| **Image** | Фоновая картинка | — выберите файл |
| **Variant** | Тема: `dark` (тёмный) или `light` (светлый) | `dark` |
| **Top Spacer** | Отступ сверху в пикселях | `48` |
| **Overlay Color** | Цвет наложения поверх картинки | `#000000` |
| **Overlay Opacity** | Прозрачность наложения (0–90%) | `0` (нет наложения) |

> **Важно:** Поле **Title** = заголовок H1. Должно быть заполнено на каждой странице для SEO.

---

### 🎰 Offer List (Список казино-офферов)

**Что делает:** Отображает список карточек казино. Может брать данные из базы (казино-офферов) или из атрибутов блока.

| Параметр | Описание | Пример |
|----------|----------|--------|
| **Title** | Заголовок секции | `CASAS COM LICENÇA` |
| **Intro** | Вступительный текст под заголовком | `Descobre os melhores...` |
| **Source** | Источник данных: `query` (из базы) или `attributes` (ручной) | `query` |
| **Limit** | Максимум офферов для показа | `15` |
| **Order By** | Порядок сортировки: `menu_order`, `rating`, `bonus`, `date`, `title` | `menu_order` |
| **Order** | Направление: `ASC` (по возрастанию) или `DESC` (по убыванию) | `ASC` |
| **Show Legal Label** | Показывать ли заголовок секции | включено |

> **Рекомендация:** Оставьте `source = query` — тогда список управляется через раздел Casino Offers в админке.

---

### 🃏 Offer Card (Карточка казино)

**Что делает:** Отдельная карточка казино (используется внутри Offer List или самостоятельно).

| Параметр | Описание |
|----------|----------|
| **Name** | Название казино |
| **Logo** | Логотип |
| **Rating** | Рейтинг (0–5, с десятичной точкой, например `4.8`) |
| **Bonus** | Текст бонуса |
| **Features** | Список преимуществ (каждое с новой строки) |
| **Payments** | Иконки методов оплаты |
| **Offer URL** | Ссылка «Jogar» (аффилиат-ссылка) |
| **Review URL** | Ссылка «Análise» (страница обзора) |

> **Совет:** Для главной страницы управляйте офферами через раздел **Casino Offers** в меню, а не через этот блок напрямую.

### 📝 Two Column Text (Текст в две колонки)

**Что делает:** Текстовая секция с заголовком и двумя колонками текста. Используется для SEO-текстов и статей.

| Параметр | Описание |
|----------|----------|
| **Title** | Заголовок секции H2 (можно использовать HTML: `<br>`, `<span>`) |
| **Background** | Цвет фона секции (например, `#f2f2f3`) |
| **Items** | Список блоков с текстом |

Каждый **Item** в списке содержит:

| Поле | Описание |
|------|----------|
| **Heading** | Подзаголовок H3 внутри секции |
| **Left** | Текст левой колонки (поддерживает HTML ссылки) |
| **Right** | Текст правой колонки (поддерживает HTML ссылки) |

### ✅ About List (Список с иконками)

**Что делает:** Секция со списком пунктов, каждый с зелёной галочкой, заголовком и описанием. Используется для «Как мы оцениваем казино» и похожих разделов.

| Параметр | Описание |
|----------|----------|
| **Title** | Заголовок секции H2 |
| **Background** | Цвет фона |
| **Items** | Список пунктов |

Каждый **Item**:

| Поле | Описание |
|------|----------|
| **Title** | Заголовок пункта (жирный текст) |
| **Description** | Описание пункта (поддерживает HTML ссылки) |

---

### 🗃️ Info Grid (Информационная сетка)

**Что делает:** Блоки-ячейки с текстом и необязательной ссылкой «Узнать больше». Используется для объяснения терминов (RTP, Rollover, и т.д.).

| Параметр | Описание |
|----------|----------|
| **Title** | Заголовок секции H2 |
| **Background** | Цвет фона |
| **Items** | Список ячеек |

Каждый **Item**:

| Поле | Описание |
|------|----------|
| **Text** | Текст ячейки (поддерживает HTML) |
| **Link Label** | Текст ссылки (например, `Mais Sobre o RTP >`) |
| **Link URL** | Адрес ссылки |

> Если **Link Label** пустой — ссылка не отображается.

### 🔗 Related Links (Связанные ссылки / карточки)

**Что делает:** Секция с карточками-ссылками на другие разделы сайта. Три варианта отображения.

| Параметр | Описание |
|----------|----------|
| **Title** | Заголовок секции H2 |
| **Intro Text** | Вступительный текст под заголовком |
| **Variant** | Вариант отображения: `cards` (карточки), `grid` (сетка), `footer` (компактный список) |
| **Background** | Цвет фона |
| **Items** | Список ссылок |

Каждый **Item**:

| Поле | Описание |
|------|----------|
| **Image** | Иконка/картинка карточки |
| **Title** | Заголовок карточки |
| **Text** | Описание (для варианта `cards`) |
| **URL** | Адрес ссылки |
| **Link Label** | Текст кнопки (по умолчанию: «Ler Mais») |

#### Варианты отображения

- **cards** — карточки с изображением, заголовком, текстом и кнопкой
- **grid** — компактная сетка (только изображение и заголовок)
- **footer** — минималистичный горизонтальный список для нижних секций

---

### 📊 Data Table (Таблица)

**Что делает:** Адаптивная таблица. На мобильных устройствах может свёртываться в карточки.

| Параметр | Описание |
|----------|----------|
| **Title** | Заголовок таблицы H2 |
| **Background** | Цвет фона |
| **Mobile Behavior** | `scroll` — горизонтальный скролл; `stacked` — карточки |
| **Headers** | Заголовки столбцов (массив строк) |
| **Rows** | Строки таблицы (массив массивов) |
| **Highlighted Rows** | Номера строк для выделения (нумерация с 0) |
| **Highlighted Cells** | Ячейки для выделения в формате «строка:столбец» |

#### Как редактировать таблицу в редакторе

В правой панели блока есть две секции:

1. **Заголовки (Headers)** — кнопки для добавления/удаления столбцов
2. **Строки (Rows)** — кнопки ★ (выделить строку), ✕ (удалить строку), **+ Add Row** (добавить строку)
3. **Bulk Import** — вставить сразу все данные в формате `Столбец 1 | Столбец 2 | Столбец 3`

#### Пример для Bulk Import

```
Casino | RTP | Destaque
Betano | 98.90% | JetX
Solverde | 97.77% | White Rabbit
```

### 📰 News Slider (Слайдер новостей)

**Что делает:** Горизонтальный слайдер с карточками статей/новостей.

| Параметр | Описание |
|----------|----------|
| **Title** | Заголовок секции | `Destaques` |
| **Enable Slider** | Включить/выключить слайдер |
| **Source** | `query` — автоматически из статей; `manual` — ручной список |

**Если Source = query:**

| Параметр | Описание |
|----------|----------|
| **Post Type** | Тип записей (обычно `post`) |
| **Category Slug** | Фильтр по рубрике (например, `casino`) |
| **Number of Posts** | Количество статей (1–20) |

**Если Source = manual:**  
Заполните список **Items** вручную. Каждый item:

| Поле | Описание |
|------|----------|
| **URL** | Адрес статьи |
| **Image** | Картинка |
| **Title** | Заголовок |
| **Date** | Дата (например, `20 Maio 2026`) |
| **Category** | Рубрика |

### ⚽ Live Odds (Виджет ставок)

**Что делает:** Слайдер с актуальными спортивными прогнозами/ставками. Данные вводятся вручную.

| Параметр | Описание |
|----------|----------|
| **Title** | Заголовок виджета |
| **Enable Slider** | Включить/выключить слайдер |
| **Provider Logo** | Путь к логотипу букмекера |
| **Provider Logo Alt** | Альтернативный текст логотипа |
| **Default CTA URL** | Ссылка по умолчанию для кнопки «перейти» |
| **CTA Label** | Текст кнопки под виджетом |
| **CTA URL** | Адрес кнопки под виджетом |
| **Items** | Список матчей/прогнозов |

Каждый **Item** (матч):

| Поле | Описание | Пример |
|------|----------|--------|
| **Team A** | Первая команда | `Benfica` |
| **Team B** | Вторая команда | `Porto` |
| **League** | Лига/турнир | `Liga Portugal` |
| **Event Date** | Дата и время в формате ISO | `2026-06-10T20:00:00Z` |
| **Market** | Рынок ставки | `Ambas Marcam: Sim` |
| **Odd** | Коэффициент | `1.85` |
| **CTA URL** | Индивидуальная ссылка (если отличается от Default) | — |
| **Is Live Updates** | Матч с Live-обновлениями | ✓/✗ |

> **Важно:** Матчи с прошедшими датами автоматически скрываются на сайте. Обновляйте список актуальными матчами.

> **Формат даты:** `2026-06-10T20:00:00Z` = 10 июня 2026, 20:00 UTC. Для португальского времени (UTC+1 летом) вычтите 1 час: `20:00 по Лиссабону = 19:00Z`.

## 10. Частые задачи — пошаговые инструкции

### Как изменить бонус у казино

1. Левое меню → **Casino Offers**
2. Найдите нужное казино, нажмите **Редактировать**
3. В блоке «Casino Offer Details» найдите поле **Bonus**
4. Измените текст
5. Нажмите **Обновить**

Изменение отразится на сайте немедленно.

---

### Как поменять порядок казино в списке

1. **Casino Offers** → откройте нужный оффер
2. В правой колонке найдите **Атрибуты страницы → Порядок**
3. Введите число (1 = первое место, 2 = второе и т.д.)
4. **Обновить**

---

### Как добавить новую статью в слайдер (при ручном режиме)

1. **Внешний вид → Редактор → Шаблоны → Главная страница**
2. Прокрутите до блока **News Slider**
3. Кликните на него
4. В правой панели, в секции **Manual items** нажмите **+ Add Item**
5. Заполните поля: URL, Image, Title, Date, Category
6. Нажмите **Сохранить** (синяя кнопка вверху)

---

### Как обновить матчи в Live Odds

1. **Внешний вид → Редактор → Шаблоны → Главная страница**
2. Кликните на блок **Live Odds** (секция «Apostas Desportivas em Destaque»)
3. В правой панели найдите секцию **Items**
4. Для устаревших матчей нажмите **Remove** и добавьте новые через **+ Add Item**
5. Заполните данные матча (команды, лига, дата, коэффициент)
6. **Сохранить**

---

### Как добавить новый пункт в меню

1. **Внешний вид → Редактор → Части шаблона → Header**
2. Кликните на блок шапки
3. В правой панели откройте **Navigation Menu**
4. Прокрутите до нужного пункта верхнего уровня → откройте его **Children**
5. Нажмите **Add sub-item** или **Add item** для пункта верхнего уровня
6. Заполните **Label** и **URL**
7. **Сохранить**

---

### Как изменить заголовок H1 на главной странице

1. **Внешний вид → Редактор → Шаблоны → Главная страница**
2. Кликните на первый блок с изображением (блок **Hero**)
3. В правой панели найдите поле **Title**
4. Измените текст
5. **Сохранить**

---

### Как изменить текст Disclaimer в подвале

1. **Внешний вид → Редактор → Части шаблона → Footer**
2. Кликните на блок подвала
3. В правой панели найдите поле **Disclaimer text**
4. Измените текст
5. **Сохранить**

---

### Как поменять логотип

1. **Внешний вид → Редактор → Части шаблона → Header**
2. Кликните на блок шапки
3. В правой панели → **Logo → Dark logo** → нажмите кнопку выбора
4. Загрузите новый файл или выберите из медиатеки
5. Повторите для **Light logo**
6. **Сохранить**

---

### Как добавить новую строку в таблицу

1. Откройте страницу с таблицей в Редакторе сайта
2. Кликните на блок **Data Table**
3. В правой панели нажмите **+ Add Row**
4. Заполните ячейки (или используйте **Bulk Import** для быстрой вставки)
5. **Сохранить**

## Полезные советы

### Цвета фона блоков

Стандартные цвета темы, которые используются в блоках:

| Цвет | Код | Применение |
|------|-----|------------|
| Белый | (пусто) | По умолчанию, прозрачный |
| Светло-серый | `#f2f2f3` | Чередование секций |

### Форматирование текста

В полях, поддерживающих HTML, можно использовать:
- `<strong>жирный текст</strong>`
- `<a href="/путь/">ссылка</a>`
- `<br>` — перенос строки
- `<span class="text--main-color">зелёный текст</span>`

### Внутренние и внешние ссылки

- **Внутренние страницы:** `/betano-portugal/` (начинайте со слэша)
- **Внешние сайты:** `https://example.com/` (полный адрес)
- **Аффилиат-ссылки:** всегда полный адрес с `https://`

### Когда изменения видны на сайте

- **Casino Offers** → сразу после нажатия **Обновить**
- **Статьи** → сразу после нажатия **Опубликовать/Обновить**
- **Редактор сайта** → после нажатия **Сохранить** (синяя кнопка)
