1. Вход в панель управления

Адрес: https://ВАШ_ДОМЕН/wp-admin/

Введите логин и пароль. После входа откроется Панель управления WordPress.

Если забыли пароль, нажмите Забыли пароль? на странице входа и укажите рабочий email.

3. Редактор сайта

Путь: Внешний вид -> Редактор

Редактор сайта используется для блоковой темы: здесь редактируются шаблоны, главная страница, шапка и подвал.

Что есть в редакторе

Редактор сайта
├── Шаблоны
│   ├── Главная страница (Front Page)
│   ├── Страница (Page)
│   ├── Запись (Single)
│   ├── Архив (Archive)
│   └── Index / fallback
└── Части шаблона
    ├── Header
    └── Footer

Как открыть нужный шаблон

  1. Откройте Внешний вид -> Редактор.
  2. Выберите Шаблоны или Части шаблона.
  3. Нажмите на нужный шаблон.
  4. Кликните по блоку на странице.
  5. Измените поля в правой панели.
  6. Нажмите Сохранить в правом верхнем углу.

Изменения в редакторе сайта видны только после нажатия Сохранить.

Важное про сохранённые шаблоны

WordPress может сохранить изменённый шаблон в базе данных. Такой шаблон перекрывает файл темы. Если разработчик обновил файл templates/front-page.html, но сайт всё ещё показывает старую версию, значит в WordPress может быть сохранён override шаблона.

Для редактора это означает простое правило: если вы меняли шаблон через редактор сайта, ваши изменения имеют приоритет над файлом темы.

4. Шапка сайта

Путь: Внешний вид -> Редактор -> Части шаблона -> Header

Header сейчас состоит из двух блоков:

  1. Возрастное предупреждение (Age Disclaimer) — верхняя полоса 18+.
  2. Шапка сайта (Site Header) — логотип и навигация.

Путь: Внешний вид -> Редактор -> Шаблоны -> Главная страница

Текущая главная страница собирается из блоков и использует общие части шаблона Header и Footer.

Структура главной

Header template part
└── Age Disclaimer
└── Site Header

Page Main
├── Hero
├── Offer List
├── Two Column Text
├── About List
├── Related Links
├── Live Odds
├── Data Table
├── Info Grid
└── News Slider

Footer template part
└── Site Footer

Главное изменение по офферам

Блок Offer List на главной сейчас настроен как source = query. Это значит, что карточки казино берутся из раздела Казино-офферы, а не редактируются вручную внутри блока.

Для изменения названия, логотипа, бонуса, рейтинга или ссылок казино используйте раздел Казино-офферы в левом меню WordPress.

7. Офферы казино

Путь: левое меню -> Казино-офферы / Casino Offers

Это централизованный список казино. Блок Offer List может автоматически брать отсюда данные.

Как добавить новый оффер

  1. Откройте Казино-офферы.
  2. Нажмите Добавить новый.
  3. В поле заголовка введите название казино.
  4. Заполните блок Детали казино-оффера.
  5. В правой панели задайте Порядок: меньшее число показывается выше.
  6. Нажмите Опубликовать.

Поля оффера

ПолеЧто вводить
ЛоготипВыберите изображение из Медиафайлов
РейтингЧисло от 0 до 5, например 4.8
БонусТекст бонуса
ОсобенностиПреимущества, по одному пункту в строке
Иконки платежейНесколько изображений через Медиафайлы
URL оффераСсылка для CTA-кнопки
URL обзораСсылка на обзор казино
Статус лицензииТекст лицензии или статуса
Легальный офферВключите чекбокс, если оффер легальный

Сортировка офферов

Если блок Offer List настроен на:

НастройкаКак работает
Order By = menu_orderСортирует по полю Порядок
Order = ASCМеньшие числа идут выше
Limit = 15Показывает максимум 15 офферов

Как скрыть оффер

  1. Откройте нужный оффер.
  2. В правой панели измените статус на Черновик.
  3. Нажмите Обновить.

Черновики не показываются в автоматическом списке.

8. Статьи и новости

Путь: левое меню -> Записи

Как опубликовать статью

  1. Нажмите Добавить новую запись.
  2. Введите заголовок.
  3. Добавьте текст и блоки статьи.
  4. В правой панели заполните:
  • Изображение записи — картинка для карточек и слайдера;
  • Рубрики — категория статьи;
  • Метки — теги, если нужны.
  1. Нажмите Опубликовать.

News Slider

Блок News Slider может работать в двух режимах:

РежимЧто происходит
queryАвтоматически показывает опубликованные записи
manualПоказывает вручную заданные элементы

Если выбран query, новые опубликованные статьи появятся автоматически, если подходят под настройки блока. Если выбран manual, элементы нужно добавлять в самом блоке.

10. Частые задачи

Изменить бонус у казино

  1. Откройте Казино-офферы.
  2. Найдите нужное казино.
  3. Откройте его на редактирование.
  4. В блоке Детали казино-оффера измените Бонус.
  5. Нажмите Обновить.

Если главная использует Offer List -> source = query, изменение появится в списке автоматически.

Поменять порядок казино

  1. Откройте нужный оффер.
  2. В правой панели найдите поле Порядок.
  3. Введите число: 1 для первого места, 2 для второго и так далее.
  4. Нажмите Обновить.

Порядок работает, если в блоке Offer List стоит Order By = menu_order.

Добавить логотип или иконки платежей к офферу

  1. Откройте оффер.
  2. В поле Логотип нажмите Выбрать изображение.
  3. Выберите файл из Медиафайлов или загрузите новый.
  4. В поле Иконки платежей выберите несколько изображений.
  5. Нажмите Обновить.

Изменить H1 на главной

  1. Откройте Внешний вид -> Редактор -> Шаблоны -> Главная страница.
  2. Кликните на блок Hero.
  3. Измените заголовок прямо в блоке или в правой панели.
  4. Нажмите Сохранить.

Заменить фоновую картинку Hero

  1. Откройте блок Hero.
  2. В правой панели нажмите Заменить фоновое изображение.
  3. Выберите картинку из Медиафайлов.
  4. Нажмите Сохранить.

Добавить пункт меню

  1. Откройте Внешний вид -> Редактор -> Части шаблона -> Header.
  2. Кликните на блок Шапка сайта.
  3. В правой панели откройте Навигационное меню.
  4. Нажмите Add item для верхнего уровня или Add sub-item внутри существующего пункта.
  5. Заполните Метка и URL.
  6. Нажмите Сохранить.

Изменить текст в подвале

  1. Откройте Внешний вид -> Редактор -> Части шаблона -> Footer.
  2. Кликните на блок Футер сайта.
  3. Откройте секцию Текст дисклеймера.
  4. Измените текст.
  5. Нажмите Сохранить.

Добавить строку в таблицу

  1. Откройте страницу с блоком Data Table.
  2. Кликните по таблице.
  3. Нажмите + Добавить строку в области таблицы или добавьте строку в поле Строки.
  4. Заполните ячейки.
  5. Нажмите Сохранить.

Обновить ручной News Slider

  1. Откройте страницу с блоком News Slider.
  2. Убедитесь, что Контент source = manual.
  3. Откройте Элементы новостей.
  4. Добавьте или измените карточки.
  5. Нажмите Сохранить.

Перевести News Slider на автоматические статьи

  1. Откройте блок News Slider.
  2. В поле Контент source выберите Запрос записей.
  3. Укажите Тип записи: обычно post.
  4. При необходимости укажите Slug категории.
  5. Выберите количество записей.
  6. Нажмите Сохранить.

Обновить матчи в Live Odds

  1. Откройте страницу с блоком Live Odds.
  2. Кликните по блоку.
  3. В правой панели откройте Прогнозы.
  4. Удалите устаревшие элементы или добавьте новые.
  5. Для даты используйте ISO UTC формат, например 2026-06-10T19:00:00Z.
  6. Нажмите Сохранить.

11. Полезные правила

Картинки и логотипы

В новых полях темы изображения выбираются через Медиафайлы. Это касается:

  • логотипов header/footer;
  • Hero background;
  • логотипов офферов;
  • иконок платежей;
  • изображений в Related Links, News Slider, Link Card и Content Section;
  • флагов во внешних ссылках футера.

Старые относительные пути вида assets/img/... всё ещё поддерживаются для импортированного контента, но редакторам лучше использовать Медиафайлы.

Ссылки

Тип ссылкиФормат
Внутренняя страница/betano-portugal/
Якорь или пустой родительский пункт меню#
Внешний сайтhttps://example.com/

HTML в текстовых полях

В полях, которые поддерживают HTML, можно использовать:

<strong>жирный текст</strong>
<a href="/page/">ссылка</a>
<br>
<span class="text--main-color">зелёный текст</span>

Не вставляйте скрипты, iframe и сложную вёрстку в текстовые поля.
