📋 Руководство редактора контента

Полный гайд по управлению сайтом в WordPress — от входа до редактирования каждого блока.

Для менеджеров WordPress Block Editor Версия 2026-06

1 Вход в панель управления

Адрес входа: https://ВАШ_ДОМЕН/wp-admin/

Введите логин и пароль, которые вам выдали. После входа откроется главная панель управления.

📸

СКРИНШОТ 1 — Страница входа в WordPress

Откройте /wp-admin/ в браузере.
Сделайте скриншот всей страницы с формой входа (два поля: логин и пароль, кнопка «Войти»).

💡

Забыли пароль? Нажмите ссылку «Забыли пароль?» под кнопкой входа. Введите email — придёт письмо для сброса пароля.

📸

Сделайте скриншот и обозначьте стрелками или рамками в левом меню:
«Записи», «Медиафайлы», «Casino Offers», «Внешний вид».

В левом меню находятся все разделы управления сайтом:

3 Редактор сайта — главный инструмент

Внешний вид → Редактор — это место, где вы редактируете дизайн и контент страниц, шапки и подвала сайта.

  1. Откройте редактор

    Левое меню → Внешний вид → Редактор

  2. Выберите, что редактировать

    В левой панели редактора выберите Шаблоны (страницы) или Части шаблона (шапка, подвал)

  3. Нажмите на нужный шаблон

    Откроется визуальный редактор с предпросмотром страницы

  4. Кликните на блок для редактирования

    Справа появятся его настройки

  5. Нажмите «Сохранить»

    Синяя кнопка в правом верхнем углу — изменения видны на сайте сразу

📸

СКРИНШОТ 3 — Интерфейс Редактора сайта

Откройте Внешний вид → Редактор → Шаблоны → Главная страница.
Сделайте скриншот всего окна браузера.
Обозначьте стрелками или рамками 4 зоны:
① Левая панель (список шаблонов)
② Центральная область (предпросмотр)
③ Правая панель (настройки блока)
④ Синяя кнопка «Сохранить» вверху справа

4 Как редактировать Шапку сайта (Header)

Внешний вид → Редактор → Части шаблона → Header
📸

СКРИНШОТ 4 — Header открыт в редакторе, блок выделен

Откройте Части шаблона → Header, кликните на шапку.
Сделайте скриншот: справа должна быть открыта панель с вкладками «Logo» и «Navigation Menu».

🖼️ Логотипы

ПолеОписание
Dark logoТёмный логотип — на светлом фоне
Light logoСветлый логотип — на тёмном фоне (в шапке)
Alt textТекст-описание логотипа для поисковиков (например, «Aposta Legal»)
💡

Чтобы поменять логотип — нажмите кнопку выбора изображения. В открывшемся окне выберите файл из медиатеки или загрузите новый (кнопка «Загрузить файл» вверху).

📸

СКРИНШОТ 5 — Панель логотипа (правая панель)

Кликните на шапку → в правой панели откройте раздел Logo.
Сделайте скриншот правой панели с кнопками выбора логотипа.

🧭 Навигационное меню

Меню имеет 3 уровня вложенности: пункты верхнего уровня → подпункты → под-подпункты.

📸

СКРИНШОТ 6 — Редактор пунктов меню

Кликните на шапку → правая панель → раздел Navigation Menu.
Разверните любой пункт меню (например, «Bónus»).
Сделайте скриншот с видимыми полями: Label, URL, Badge text, кнопкой «Add sub-item».

Каждый пункт меню содержит:

ПолеОписаниеПример
LabelТекст пунктаBónus
URLСсылка (внутренняя или внешняя)/bonus-em-portugal/
IconПуть к иконке (оставить как есть если не меняете)assets/img/.../icon.svg
Badge textБейдж на пункте менюnovo
Badge bg colorЦвет фона бейджа#12a96a
New badgeВключить/выключить бейдж «NEW»✓ или ✗
Mega menuИспользовать широкое меню для этого пункта✓ для «Recursos»
ℹ️

Правила для URL:
• Страницы этого сайта: /betano-portugal/ (начинается со слэша)
• Внешние сайты: https://example.com/ (полный адрес)
• Якорь (не-ссылка): #

7 Управление Офферами казино

Casino Offers → Все Casino Offers

Это список всех казино, которые отображаются на главной странице. Данные редактируются здесь — не в Редакторе сайта.

📸

СКРИНШОТ 9 — Список Casino Offers в админке

Левое меню → Casino Offers.
Сделайте скриншот списка со всеми казино. Обозначьте колонку «Порядок» — она определяет позицию в списке.

Как добавить новое казино

  1. Нажмите «Добавить новый Casino Offer»

    Кнопка вверху страницы Casino Offers

  2. Введите название казино

    В поле заголовка вверху страницы (например, «Betano»)

  3. Заполните блок «Casino Offer Details»

    Все поля описаны в таблице ниже

  4. Задайте порядок

    Правая панель → Атрибуты страницы → Порядок (1 = первое в списке)

  5. Нажмите «Опубликовать»

    Казино появится на сайте

📸

СКРИНШОТ 10 — Страница редактирования Casino Offer

Откройте любой оффер для редактирования.
Сделайте скриншот блока «Casino Offer Details» с полями.
Обозначьте стрелкой поле «Порядок» в правой панели.

Поля Casino Offer Details

ПолеОписаниеПример
LogoЛоготип казино — выберите картинку из медиатекиbetano-logo.svg
RatingРейтинг от 0 до 5 (с точкой, не с запятой)4.8
BonusТекст бонуса100% até 500€
FeaturesПреимущества — каждое с новой строкиLevantamentos rápidos

Offer URLАффилиат-ссылка для кнопки «Jogar»https://apostalegal.pt/ir/betano

⚠️

Рейтинг вводите с точкой, не с запятой: 4.8 — правильно, 4,8 — неправильно.

Как изменить порядок казино

  1. Откройте нужный оффер

    Casino Offers → кликните на название

  2. Найдите «Порядок» в правой панели

    Атрибуты страницы → Порядок

  3. Введите число

    1 = первое место, 2 = второе и т.д.

  4. Нажмите «Обновить»

    Порядок на сайте изменится сразу

Как временно скрыть казино с сайта

Откройте оффер → правая панель → Статус: Черновик → Обновить. Казино пропадёт с сайта, данные сохранятся.

8 Статьи и Новости

Записи → Все записи
📸

СКРИНШОТ 11 — Редактор записи (статьи)

Откройте любую статью или создайте новую (Записи → Добавить новую запись).
Сделайте скриншот редактора с видимой правой панелью.
Обозначьте: поле заголовка, область текста, «Изображение записи», «Рубрики».

  1. Создайте новую запись

    Записи → Добавить новую запись

  2. Введите заголовок

    Кликните в поле «Добавить заголовок» вверху

  3. Напишите текст

    Кликните ниже и начните печатать. Нажмите / чтобы вставить блок (заголовок, картинка, список и т.д.)

  4. Добавьте главное изображение

    Правая панель → Изображение записи → Задать изображение записи

  5. Выберите рубрику

    Правая панель → Рубрики → отметьте нужную

  6. Опубликуйте

    Синяя кнопка «Опубликовать» вверху справа

News Slider — Слайдер статей

new-theme/news-slider

Горизонтальный слайдер с карточками статей. Два режима: автоматический (берёт статьи из WordPress) и ручной (список задаётся вручную).

ПараметрОписание
TitleЗаголовок секции
Enable SliderВключить/выключить слайдер
Sourcequery — автоматически из статей
manual — ручной список

Если Source = query (автоматический):

ПараметрОписание
Post TypeТип записей (оставьте post)
Category SlugФильтр по рубрике (например, casino). Пустое = все рубрики
Number of PostsКоличество статей (1–20)

Если Source = manual (ручной) — каждый Item:

ПолеОписание
URLАдрес статьи
ImageГлавное изображение
TitleЗаголовок
DateДата (например, 20 Maio 2026)
CategoryНазвание рубрики
⚽

Live Odds — Виджет ставок

new-theme/live-odds

Слайдер с актуальными спортивными прогнозами/ставками. Данные вводятся вручную. Матчи с прошедшими датами автоматически скрываются.

📸

СКРИНШОТ — Live Odds виджет на сайте

Сделайте скриншот виджета Live Odds на главной странице. Обозначьте: заголовок, карточку матча (команды, лига, дата, коэффициент, кнопка), логотип букмекера.

ПараметрОписание
TitleЗаголовок виджета
Enable SliderВключить/выключить слайдер
Provider LogoПуть к логотипу букмекера
Default CTA URLСсылка по умолчанию для всех матчей
CTA LabelТекст кнопки внизу виджета
CTA URLСсылка кнопки внизу виджета
ItemsСписок матчей

Каждый матч (Item):

ПолеОписаниеПример
Team AПервая командаBenfica
Team BВторая командаPorto
LeagueЛига / турнирLiga Portugal
Event DateДата и время матча (формат ISO 8601)2026-06-10T20:00:00Z
MarketРынок ставкиAmbas Marcam: Sim
OddКоэффициент1.85
CTA URLИндивидуальная ссылка (если другая)—
Is Live UpdatesМатч с Live-обновлениями✓ / ✗
⏰

Формат даты: 2026-06-10T20:00:00Z
Расшифровка: год-месяц-деньTчас:минута:секундаZ
«Z» в конце = время UTC. Португалия летом = UTC+1, поэтому:
матч в 21:00 по Лиссабону = 2026-06-10T20:00:00Z

⚠️

Матчи с прошедшими датами автоматически скрываются на сайте. Обновляйте список актуальными матчами — иначе виджет не будет виден посетителям.

✔ Частые задачи — пошаговые инструкции

🎰 Изменить бонус у казино

~1 минута
  1. Casino Offers

    Левое меню → Casino Offers

  2. Найдите казино и откройте

    Кликните на название

  3. Измените поле Bonus

    В блоке «Casino Offer Details»

  4. Нажмите «Обновить»

    Изменение видно на сайте сразу

🔢 Поменять порядок казино в списке

~2 минуты
  1. Casino Offers → откройте нужный оффер
  2. Правая панель → Атрибуты страницы → Порядок

    Введите число (1 = первое, 2 = второе...)

  3. «Обновить»
💡

Чтобы поставить казино первым — введите 1. Второе казино должно иметь 2, третье — 3 и т.д.

⚽ Обновить матчи в виджете Live Odds

~5 минут
  1. Редактор → Шаблоны → Главная страница
  2. Кликните на блок Live Odds

    Секция «Apostas Desportivas em Destaque»

  3. В правой панели → Items

    Нажмите Remove для устаревших матчей

  4. Нажмите «+ Add Item»

    Заполните: Team A, Team B, League, Event Date, Market, Odd

  5. «Сохранить»

📝 Добавить статью в слайдер «Destaques» (ручной режим)

~3 минуты
  1. Редактор → Шаблоны → Главная страница
  2. Прокрутите вниз до News Slider

    Кликните на него

  3. Правая панель → Manual items → «+ Add Item»
  4. Заполните поля

    URL, Image, Title, Date, Category

  5. «Сохранить»

📋 Добавить строку в таблицу

~2 минуты
  1. Редактор → Шаблоны → Главная страница
  2. Кликните на нужную таблицу
  3. Правая панель → «+ Add Row»
  4. Заполните ячейки

    Или используйте Bulk Import: вставьте строки через |

  5. «Сохранить»

🧭 Добавить пункт в навигационное меню

~3 минуты
  1. Редактор → Части шаблона → Header
  2. Кликните на шапку
  3. Правая панель → Navigation Menu
  4. Найдите нужный раздел и нажмите «Add sub-item»
  5. Заполните Label и URL
  6. «Сохранить»

📰 Опубликовать новую статью

~5 минут
  1. Записи → Добавить новую запись
  2. Введите заголовок
  3. Напишите текст

    Нажмите / чтобы вставить блок (заголовок, картинка, список)

  4. Добавьте главное изображение

    Правая панель → Изображение записи

  5. Выберите рубрику

    Правая панель → Рубрики

  6. «Опубликовать»

🖼️ Заменить логотип в шапке

~2 минуты
  1. Редактор → Части шаблона → Header
  2. Кликните на шапку → правая панель → Logo
  3. Dark logo → кнопка выбора → загрузите файл
  4. Повторите для Light logo
  5. «Сохранить»

📊 Скрыть казино без удаления

~1 минута
  1. Casino Offers → откройте нужное казино
  2. Правая панель → Статус → «Черновик»
  3. «Обновить»

    Казино исчезнет с сайта. Данные сохранятся.

🔗 Правила для ссылок

✅ Правильно

/betano-portugal/ — внутренняя
https://example.com/ — внешняя
# — якорь/заглушка

❌ Неправильно

betano-portugal — без слэша
example.com — без https://
localhost.com/... — тестовый URL

📝 HTML в текстовых полях

Некоторые поля поддерживают HTML-теги:

<strong>текст</strong>
→ жирный текст
<a href="/путь/">текст</a>
→ ссылка
<br>
→ перенос строки
<span class="text--main-color">текст</span>
→ зелёный текст
